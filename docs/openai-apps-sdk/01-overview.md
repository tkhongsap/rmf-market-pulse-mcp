# Apps SDK Overview

**Source:** https://developers.openai.com/apps-sdk

## Overview
The Apps SDK is OpenAI's framework for building applications that extend ChatGPT. It's currently in preview for developers to test, with "app submission opening later this year."

## Key Sections

**Core Components:**
- MCP Server setup and configuration
- User interaction patterns
- Design guidelines native to ChatGPT

**Development Workflow:**
The documentation organizes development into three phases:
1. **Plan** - Research use cases and define tools
2. **Build** - Set up servers, create custom interfaces, handle authentication
3. **Deploy** - Launch apps and connect them to ChatGPT

**Important Resources:**
- App developer guidelines covering quality and safety standards
- Design guidelines for conversational flows
- Security and privacy considerations
- Troubleshooting documentation
- Metadata optimization guidance
