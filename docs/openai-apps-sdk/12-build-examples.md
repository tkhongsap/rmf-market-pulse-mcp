# Examples

## Overview

The documentation describes the Pizzaz demo application as a bundled collection of UI components designed to illustrate the complete tool surface area in an end-to-end format. The guide walks developers through the MCP server architecture and component implementations that enable those tools.

## Getting Started

The page indicates that "these examples serve as blueprints when you assemble your own app." Developers can access the Pizzaz demo and additional examples in the [OpenAI Apps SDK Examples repository on GitHub](https://github.com/openai/openai-apps-sdk-examples).
