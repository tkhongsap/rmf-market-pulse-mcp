# Apps SDK - OpenAI Developers

## Overview

"Our framework to build apps for ChatGPT." The Apps SDK is currently available in preview for developers to begin building and testing applications. Full app submission capabilities will open later this year.

## Getting Started

### Launch Your First App
Follow the Apps SDK quickstart guide to build and connect an initial app to ChatGPT.

### Plan Phase
Research and identify viable Apps SDK use cases, then prioritize which directions to pursue.

### Build Phase
Wire together tools, templates, and the widget runtime that powers ChatGPT applications.

### Deploy Phase
Learn deployment procedures for your MCP server infrastructure.

## Design for Quality

### App Design Guidelines
The framework emphasizes creating components and conversational flows that integrate naturally within ChatGPT's interface.

### App Developer Guidelines
Applications must meet established quality, safety, and policy standards set by OpenAI.
