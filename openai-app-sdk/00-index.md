# Apps SDK

## Overview

The Apps SDK is OpenAI's framework for building applications that extend ChatGPT's capabilities. Currently available in preview, it allows developers to create and test apps with plans for opening submissions later this year.

## Getting Started

### Quick Launch
Follow the Apps SDK quickstart to build and connect your first app to ChatGPT.

### Three Core Phases

**Plan**: Research use cases and identify Apps SDK opportunities for your application.

**Build**: Create and configure an MCP server as the foundation for your app.

**Deploy**: Learn deployment procedures for your MCP server infrastructure.

## Design & Quality Standards

### App Design Guidelines
"Design components and conversational flows that feel native to ChatGPT." These guidelines ensure user experiences align seamlessly with the platform's interface and interaction patterns.

### Developer Guidelines
Apps must meet OpenAI's quality, safety, and policy standards before submission consideration.
